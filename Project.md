Приложение получает на входе несколько картинок (загружается) для выставки
по соответствующей кнопке, 
подключает AI и генерирует описания каждой.
На основе совокупного анализа предлагает 3 варианта названия выставки для выбора.
После выбора одного из вариантов — генерирут макет афиши и кратное описние
Интерфейс:
1. Поле для загрузки картинок.
2. 3 кнопки: «Описание», «Выставка», «Афиша».
3. Блок для отображения результата, в котором
происходит генерация нужного вида ответа.
